# Olist Production Pipeline 🚀

An end-to-end production-grade ELT data pipeline for the Olist e-commerce dataset. This project automates data ingestion into **PostgreSQL**, handles complex transformations with **dbt**, and ensures data integrity using **Great Expectations**.

## 📁 Project Structure
- **ingestion/**: Python scripts for loading raw CSVs into PostgreSQL `raw` schema.
- **olist_transform/**: dbt models for cleaning, modeling, and feature engineering.
- **notebooks/**: EDA and Lorenz Curve visualization for market cluster analysis.
- **gx/**: Great Expectations configuration and data validation suites.
- **.github/workflows/**: CI/CD pipeline for automated testing and deployment.

## 🛠️ Tech Stack
- **Database**: PostgreSQL 15
- **Orchestration**: GitHub Actions (CI/CD)
- **Transformations**: dbt (Data Build Tool)
- **Data Quality**: Great Expectations (GX)
- **Environment**: Python 3.11

## ⚙️ CI/CD Pipeline
The project includes a GitHub Action (`Olist Production Pipeline`) that triggers on every push to `main`. It performs:
1. **Infrastructure**: Spins up a Dockerized PostgreSQL service.
2. **Ingestion**: Runs `load_raw.py` to populate the database.
3. **Transformation**: Executes `dbt run` and `dbt test`.
4. **Validation**: Runs a Great Expectations data quality gate.
5. **Artifacts**: Uploads dbt logs and targets for 7 days.

## 🚀 Local Getting Started

### 1. Installation
```bash
# Install core dependencies
pip install -r requirements.txt

# Install dbt-postgres adapter
pip install dbt-postgres
```

### 2. Database Configuration
Ensure your PostgreSQL credentials are set in your environment:
- **DB**: `olist_dw`
- **User**: `postgres`
- **Host**: `localhost`

### 3. Execution
```bash
# 1. Ingest
python ingestion/load_raw.py

# 2. Transform
cd olist_transform && dbt run --profiles-dir .

# 3. Validate
python run_quality_check.py
```
